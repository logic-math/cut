# 项目目标 (Objectives)

## O1: 让创作者以最低成本将文字作品转化为视听视频
描述：创作者只需写好讲稿/文章，通过 cut skills 技能包即可完成从文字到视频的全流程制作，无需手工操作每个环节。

### 关键结果 (Key Results)
- KR1.1: 输入一篇讲稿，5分钟内生成结构化视听脚本（包含分镜、旁白、画面、音乐描述）
- KR1.2: 脚本经人工确认后，自动完成素材搜索（视频/图片/音乐）并提供人工审核界面
- KR1.3: 支持视频、图片、手绘图、音频四种媒体形式的素材制作/搜索
- KR1.4: 通过 FFmpeg 将所有素材合成为最终视频文件

## O2: 构建可维护、可扩展的 Claude Code skills 技能包
描述：技能包遵循 Anthropic skills 规范，每个底层服务（TTS、AI生成等）均可替换，整体架构模块化。

### 关键结果 (Key Results)
- KR2.1: 符合 anthropics/skills SKILL.md 规范，可通过 Claude Code 直接调用
- KR2.2: TTS、AI图像生成、AI视频生成等底层服务均采用依赖倒置，可通过配置切换服务商
- KR2.3: 每个 skill 有独立的测试用例和评估框架
- KR2.4: 运行前自动检测环境依赖（FFmpeg、Python包等），缺失时给出安装提示
